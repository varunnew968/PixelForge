# iLoveImage - Premium Image Suite

iLoveImage is a state-of-the-art, privacy-focused image processing application built with React, Vite, and Tailwind CSS. All processing happens entirely in your browser, ensuring your data never leaves your device.

## ✨ Features

- **🚀 Image Compressor**: Reduce file sizes by up to 90% without quality loss.
- **📐 Image Resizer**: Batch resize with custom dimensions or popular presets (Full HD, Instagram, etc.).
- **🔄 Format Converter**: Convert between JPG, PNG, and WebP instantly.
- **✂️ Smart Crop**: Precise framing with aspect ratio locking.
- **💎 Premium UI**: Modern glassmorphism design with neon accents and smooth Framer Motion animations.
- **🔒 Privacy First**: 100% client-side processing using the HTML5 Canvas API.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite 8, Tailwind CSS 3
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Image Processing**: browser-image-compression, HTML5 Canvas API, React Image Crop

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components`: Reusable UI elements (Navbar, Footer, Dropzone).
- `src/pages`: Individual tool implementations.
- `src/layouts`: Main layout with background effects.
- `src/index.css`: Global styles, glassmorphism utilities, and neon effects.
